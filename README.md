# Calculator Web App

A beginner-friendly web app for simple and scientific calculations using HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Scientific functions (sin, cos, tan, log, sqrt, power)
- Easy to use interface

## How to Use
1. Open `index.html` in your browser.
2. Use the buttons to enter your calculation.
3. Click `=` to get the result.
4. Use scientific functions for advanced calculations.

## How to Publish on GitHub
1. Create a new repository on GitHub.
2. Upload all files (`index.html`, `style.css`, `script.js`, `README.md`).
3. Enable GitHub Pages in repository settings to publish your site.

## Project Structure
- `index.html` — Main web page
- `style.css` — Styles for the calculator
- `script.js` — Calculator logic
- `README.md` — Project instructions
